サービス概要

映画監督が開催する役者志望の方のための演技ワークショップ応募サイトです。
以前、制作したものをもとにダミーサイトを作りました。

応募者が入力した情報をデータベースに格納し
管理者ログイン画面で応募者データを閲覧できるようになっています。

セールスポイント

  • 多数の応募が想定されるためデータベースを構築しデータを格納できるようにしました。
  • 応募者の個人情報を扱うため堅牢なServlet+JSPでEclipseを使用し開発しました。
  • 監督からの要望によりフォームに応募者の顔写真画像アップロード機能をつけました。
  • 応募者が募集サイトをスマホで見ることを想定しレスポンシブルデザインにしました。
  • GCEにターミナルからJava＆Tomcatの環境構築をしDBを作成し配置しました。

使用技術

  • JavaScript
  • HTML5
  • CSS3
  • PHP
  • AdobePhotoshop
  • AdobePremierePro

使用例

  1. 申込フォームからをクリック

    2. メニューからコース紹介のお申込フォームボタンをクリックします。

  2. 申込情報を入力し応募

    3. 入力フォームに応募情報を入力します。確認ボタンを押し入力情報を確認したら送信ボタンを押します。 これでデータベースに応募情報が格納されました。

  3. 管理者ログイン画面から応募者閲覧画面へ

    4. IDとパスワードを入力しログインし応募者を閲覧できます。

改善点

  1. データベース検索機能

    2. 今回は応募者のデータを全て取得し表示するだけになっています。 なのでデータベースに格納された応募者のデータを検索し表示する機能を追加したいと考えております。

  2. データベース編集機能

    3. 応募者データに対して監督が管理画面でメモ書きやタグ付を出来る機能を追加したいと考えております。また、 書類選考で落選された応募者のデータを管理画面上で監督自身が削除できる機能も追加したいと考えております。

  3. メール送信機能

    4. 応募者が入力フォームで入力した情報が今回はデータベースに格納されるだけになっています。 なので入力情報をメールにも送信する機能も追加したいと考えております。

概要


プログラミングを学び制作したものをポートフォリオにしました。
私はお酒が好きなので、ビールをイメージしたデザインのサイトにしました。

自由度の高いデザインにしたかったためフルスクラッチでサイト構築をしました。
少ない時間で情報が伝わるようページ数を少なくしシンプルなデザインにしました。

セールスポイント

  • HTML,CSS,JavaScripを使用しフルスクラッチでサイト構築しました。
  • javaScriptでホバー時やクリック時に動きのあるサイトにしました。
  • PHPでお問い合わせフォームを構築しました。
  • インターネットサーバーと契約し独自ドメインを取得しサイトを公開しました。

使用技術

  • JavaScript
  • HTML5
  • CSS3
  • PHP
  • AdobePhotoshop
  • AdobePremierePro

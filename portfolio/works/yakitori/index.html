サービス概要

気軽に居酒屋に行けるようになりたい
その気持ちからやきとり居酒屋のウェブサイトを作りました。

定番メニューの情報を管理画面から入力しデータベースに格納できるようにしました。
データベースの情報からウェブサイトの表示が出来るようになっています。

セールスポイント

  • ウェブサイトの定番メニューを管理画面から簡単に変更できるようにしました。
  • 変更した情報をデータベースから取得しウェブサイトに表示します。
  • スマホで見ることを想定しレスポンシブルデザインにしました。
  • １カラムのシンプルなデザインにしました。
  • ウェブサーバーを契約し、PHPでデータベース接続できるようにしました。

使用技術

  • PHP
  • MySQL
  • JavaScript
  • HTML5
  • CSS3

使用例

  1. 定番メニューの変更

    2. 定番メニューの4番目にある「ピリ辛きゅうり」を変更します。

  2. 管理画面からメニューを削除

    3. 管理画面にログインし、定番メニューの「ピリ辛きゅうり」の 削除ボタンをクリックします。クリックするとデータベース からも「ピリ辛きゅうり」の情報が削除されます。

  3. 追加メニューを入力

    4. フォームから追加メニューの「料理名」「値段」「画像」 を追加し「追加」ボタンをクリックします。 するとデータベースにも追加メニューの情報が格納されます。

  4. 追加メニューの確認

    5. やきとり屋のウェブサイトの定番メニューに 先ほど追加した「卵焼き」が表示されています。

改善点

  1. 管理画面の編集機能

    2. 今回は「定番メニュー」しか管理画面で編集を出来ないので、「メニュー」や「店内のご案内」も 管理画面から編集できるようにしていきたい。

  2. データベースからの表示

    3. 「定番メニュー」の表示が4項目以上は表示するメニューの数を管理画面から決められたり、 データベースに格納されているメニューから表示非表示を選べるようにしていきたい。

サービス概要

気軽に居酒屋に行けるようになりたい
その気持ちからやきとり居酒屋のウェブサイトを作りました。

定番メニューの情報を管理画面から入力しデータベースに格納できるようにしました。
データベースの情報からウェブサイトの表示が出来るようになっています。

セールスポイント

  • ウェブサイトの定番メニューを管理画面から簡単に変更できるようにしました。
  • 変更した情報をデータベースから取得しウェブサイトに表示します。
  • スマホで見ることを想定しレスポンシブルデザインにしました。
  • １カラムのシンプルなデザインにしました。
  • ウェブサーバーを契約し、PHPでデータベース接続できるようにしました。

使用技術

  • PHP
  • MySQL
  • JavaScript
  • HTML5
  • CSS3

使用例

  1. 定番メニューの変更

    2. 定番メニューの4番目にある「ピリ辛きゅうり」を変更します。

  2. 入力フォームに応募情報を入力します。確認ボタンを押し入力情報を確認したら送信ボタンを押します。 これでデータベースに応募情報が格納されました。

  3. 管理者ログイン画面から応募者閲覧画面へ

    4. IDとパスワードを入力しログインし応募者を閲覧できます。

改善点

  1. データベース検索機能

    2. 今回は応募者のデータを全て取得し表示するだけになっています。 なのでデータベースに格納された応募者のデータを検索し表示する機能を追加したいと考えております。

  2. データベース編集機能

    3. 応募者データに対して監督が管理画面でメモ書きやタグ付を出来る機能を追加したいと考えております。また、 書類選考で落選された応募者のデータを管理画面上で監督自身が削除できる機能も追加したいと考えております。

  3. メール送信機能

    4. 応募者が入力フォームで入力した情報が今回はデータベースに格納されるだけになっています。 なので入力情報をメールにも送信する機能も追加したいと考えております。
